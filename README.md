# The Project Maneger

A local-first PWA for tracking personal projects, stakeholders, and stages — designed for visualization, fast capture, and one-click status reports.

Everything lives in your browser's IndexedDB. No accounts, no backend, no AI.

## Stack

React 19 · Vite 8 · TypeScript · Tailwind 3 · Dexie · Zustand · @dnd-kit · @uiw/react-md-editor · vite-plugin-pwa

## Develop

```bash
pnpm install
pnpm dev      # dev server
pnpm test     # vitest
pnpm build    # production build (PWA enabled)
pnpm preview  # serve the production build
```

## Design

- **Calm Glass** — desaturated palette (slate `#6b7ca8` · sand `#c4a66f` · sage `#85b89d` · rose `#9a6b71`), translucent surfaces
- **Lexend Variable** default font, **OpenDyslexic** alternative — togglable in Settings
- **Dark + light + system** themes
- Lexend is dyslexia-friendly out of the box

## Features

- Per-project Kanban with drag-and-drop and Timeline view
- Home: pinned project cards + sortable list + portfolio Timeline
- Inbox quick-capture → convert to project items
- One-click status reports (markdown + PDF)
- Settings: appearance, custom templates, JSON export/import, archived projects
- Installs as a PWA with offline support

## Data shape

Three IndexedDB tables: `projects`, `items`, `inboxEntries`. See [`src/db/schema.ts`](src/db/schema.ts).

## Reports

Generated by [`src/features/reports/generate.ts`](src/features/reports/generate.ts) — pure function, fully tested. PDF export sanitizes markdown through `marked` and `DOMPurify`, then mounts the result via `DOMParser` and `adoptNode` for a safe popup.
